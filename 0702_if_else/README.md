# 條件控制

## if...else...
- 條件式回傳bool  
```java  
if(condition){  
	do statement;  // condition true
}  
else{  
	do statement;  // condition false	  
}  
```

## if...else if... else
- 多組條件判斷
```java  
if(condition1){  
	do statement;  // condition true
}  
else if(condition2){  
	do statement;  // condition true
}  
else{  
	do statement;  // condition1 and condition2 both are false	  
}  
```
